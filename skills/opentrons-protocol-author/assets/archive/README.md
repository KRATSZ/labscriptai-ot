# Archived protocol snippets

These files are **supplementary** examples (minimal transfers, OT-2 minimal, tip-recovery reference). They are **not** the default starting points for new protocols.

**Default starters** (use these first):

- `../flex_protocol_template.py` — Flex blank
- `../protocol_template.py` — OT-2 blank
- `../flex_template_*.py` — workflow-shaped templates
- `../flex_liquid_classes_example.py` — Flex verified liquid classes

**In this folder:**

| File | Role |
|------|------|
| `flex_minimal_transfer.py` | Minimal Flex transfer skeleton |
| `ot2_minimal_transfer.py` | Minimal OT-2 transfer skeleton |
| `flex_tip_recovery_reference.py` | Reference for recovery / tip scenarios |
